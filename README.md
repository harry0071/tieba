# vue-spa

> A Vue.js project

## 用法
将项目clone到本地。然后执行下面的命令

``` bash
# 首先安装依赖
npm i

# 然后运行项目
npm start

# 发布打包
npm run build

# 或者
# 发布打包并查看分析报告
npm run build --report
```

## 自定义路径说明
```
'@': resolve('src'),
'cp': '@/components',
'css': '@/assets/css',
'js': '@/assets/js',
'imgs': '@/assets/imgs'
```
